# Flask Web App Tutorial

## Setup & Installtion

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
python main.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`


Authentication API -

    1) Signup API
    Input Parameters -
    user Name
    User Email
    User Mobile Number
    User Password

    2) Login API
    Input Parameters -
    User Email or User Mobile Number
    User Password
